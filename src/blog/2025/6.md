# Rust进阶知识学习笔记

## 什么是闭包？

闭包是一种可以捕获调用者作用域变量的匿名函数。它既可以作为值赋值给变量，又可以作为实参传递给函数，闭包让函数变成了一种数据类型的存在。

下面这是一个好例子，cat的值是一个闭包，它捕捉了变量x。

```rust
fn main() {
   let x = 100;
   let cat = |y| x + y;
   assert_eq!(101, cat(1));
}
```
